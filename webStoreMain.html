Introduction to the WebStore

This project is created as a maquette of WebStore which I want to do. It represents customers with their contact info, shipping addresses, transactions details and information about purchase.


Methodology. Technologies used.

This project was made with the following technologies:

  • XML
  • XML Schema
  • XSL
  • XSLT
  • HTML

Work content. Project files.

This project consists of the following files:

  1. XML File
  2. XSD File
  3. XSLT file
  4. HTML file

Elements of the XML

webStore is the root element, it contains 5 customer records as child elements of the webStore.

Each record of customer has:

Web Account

  • Email
  • Nickname
  • Password

Payment Info

  • Bank Card Type
  • Card No
  • Valid Thru
  • Cvv
  • Amount
  • Currency
  • Date of Payment

Shipping Information

  • Country
  • City
  • Street
  • Number
  • Zip
  • Date of Shipping

Notification About Purchase

  • Message type
  • Phone type

Items in basket

  • Title
  • Category
  • Country of Manufecture
  • Brand
  • Model
  • Year
  • Code
  • Price
  • Currency

Bought Items Info

  • Title
  • Category
  • Country of Manufecture
  • Brand
  • Model
  • Year
  • Code
  • Date of purchase
  • Price
  • Currency
